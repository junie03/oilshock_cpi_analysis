# 국제 유가 변동이 국내 물가(CPI)에 미치는 영향 분석

2026년 2월 28일 발발한 전쟁을 하나의 외부 충격(shock)으로 두고, 국제 유가 변동이 국내 유가 및 소비자물가(CPI)에 어떻게, 얼마나 시차를 두고 전이되는지를 4개의 가설로 나누어 검증한 팀 프로젝트입니다.

## 프로젝트 개요

- **분석 기간**: 2022년 1월 ~ 2026년 5월 (가설별로 상이, 각 노트북 참고)
- **핵심 사건**
  - 2026-02-28: 전쟁 발발
  - 2026-04-02: 유가 최고 충격일
  - 2026-04-08: 무기한 휴전 선포
- **데이터 출처**: 오피넷(국내 휘발유 소비자가격), WTI/Dubai/Brent(국제 유가), 통계청(소비자물가지수 CPI)
- **분석 도구**: Python(pandas, numpy, scipy, statsmodels, scikit-learn), SQLite(SQL 기반 월별/일별 집계)

## 가설 및 핵심 결론

| 구분 | 가설 | 검증 방법 | 결론 |
|------|------|-----------|------|
| **H1** | 전쟁 발생 이후 국제 유가는 통계적으로 유의미하게 상승한다 | 독립표본 t-test (전쟁 전 vs 후 국내 휘발유 가격) | 채택 — 전쟁 전 평균 1,700.17원/L → 전쟁 후 1,822.97원/L (p < 0.0001). 국제 유가 급등 후 국내 가격은 시차를 두고 반응 |
| **H2** | 유가 상승이 일정 시차를 두고 국내 CPI에 전이된다 | 전쟁 시점 기준 t-1/t0/t+1/t+2 이벤트 스터디 | 방향성 지지 — WTI→CPI 약 2개월, 국내유가→CPI 동월~1개월 시차 확인(단, N=4로 정성적 수준) |
| **H3** | 전쟁 이후 물가의 움직이는 방식(변동성·지속성) 자체가 구조적으로 달라졌다 | 국면별(안정기/충격기/고착화기) 구조 변화 및 하방경직성 정량화 | 채택 — 국제 유가는 -13.79% 하락한 반면 국내 유가는 +6.67% 상승. "오를 땐 즉시, 내릴 땐 지연"되는 하방경직성(Rockets and Feathers) 확인 |
| **H4** | 유가 변동은 미래 CPI 상승의 선행 지표로 활용 가능하다 | 교차상관분석 + AR/ADL 예측모델 비교(RMSE, MAE) | 채택 — 유가 포함 모델(Model 2)이 CPI 단독 모델(Model 1) 대비 RMSE 0.7%, MAE 4.6% 개선 |

각 노트북은 SQLite 인메모리 DB(`sqlite3`)에 데이터를 적재한 뒤 SQL로 집계하는 방식(`oil_sql` 방식)을 공통으로 사용하며, 원본 데이터는 코드 내에 직접 포함되어 있어 별도의 외부 파일 없이 실행 가능합니다.

> **참고**: 시각화 코드의 한글 폰트 설정(`Malgun Gothic` 등)은 Windows 기준입니다. macOS는 `AppleGothic`, Linux는 `NanumGothic` 등으로 실행 환경에 맞게 수정이 필요할 수 있습니다.

## 분석의 한계

- 표본 크기가 작아(N=4~52개월) 일부 가설은 통계적 추론보다 정성적 사례연구에 가까움
- 2026년 전쟁이라는 단일 사건 기반 분석으로 일반화에 한계
- 환율, 기준금리, 계절성 등 다른 혼란변수(confounding factor)를 통제하지 않음
- 국내 유가는 유류세·정유사 마진 등으로 국제 유가 대비 완충 효과가 존재해 해석 시 유의 필요
